HOW TO PLAY — MORTAL KEYS

OBJECTIVE: 10 questions per round, as few errors as possible. Leaderboard ranks by errors (fewer = better), then by time. Every type appears at most twice; exact same question never repeats.

IDENTIFICATION — question shown on the keyboard, press the correct button:

  • Note ID: One key highlighted — identify the note (2 octaves: Do3–Si4).
  • Interval ID: R = Root (green), interval key (orange). Identify the interval (m2–P8). Both notes sound on correct answer.
  • Chord ID: Close voicing shown on keyboard (all inversions). Identify the chord type. Chord plays on correct answer. Excluded chord types show ✕.
  • Scale ID: Scale notes highlighted in an octave box, R = Root (red). Keys outside the box are darkened. Identify the scale. Scale plays on correct answer. Excluded scale types show ✕.

CONSTRUCTION — target illuminated below keyboard, click keys to build it, press SUBMIT:

  • Note Construction: Place both occurrences of the target note on the keyboard (one per octave).
  • Interval Construction: R pre-placed (green) — click the nearest interval note above it. Auto-checks immediately. Tap R to hear it.
  • Chord Construction: One chord tone pre-placed (amber, with role label). Click remaining tones in any order. No duplicate keys. SUBMIT to verify.
  • Scale Construction: R pre-placed (red) in octave box. Fill all scale notes within the box. SUBMIT to verify.

OCTAVE BOX: Chord and Scale modes — choose a fixed octave range (Do–Do, Re–Re…) or 💀 KHAOS for a random octave each question.

ACTIVE CHORDS / SCALES: In the lobby, deselect chord or scale types you want to skip. Deselected types show ✕ in the lobby and during the game. Leaderboard can be filtered by configuration (ALL or SAME CONFIG).

WRONG ANSWER: Retry the same question. Timer keeps running. After 2 wrong answers ☠ MERCY activates.

MERCY: Hold ☠ to reveal the full solution. Release to hide it and retry. Resets each question.

SCORING: 0 errors = FLAWLESS VICTORY · 1–2 = Excellent · 3–5 = Skilled · 6–10 = Fight Harder · 11–19 = Finish Him · 20+ = Fatality
